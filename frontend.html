Roadmap for Frontend Developer

HTML Basics

Topics: Structure of an HTML document, basic tags ( html, head, body, p, h1 to h6, a, img ).

Topics: Forms (form, input, button, textarea ), semantic tags (header, footer, article, section ).

Task: Create a contact form with input fields and a submit button.

Task: Create a basic webpage with a title, heading, paragraph, and an image.

CSS Basics

Topics: CSS syntax, selectors, colors, and text styles.

Box Model and Positioning.

Flexbox.

CSS Projects

Topics: Apply Flexbox and box model to design a responsive webpage.

Task: Make a Responsive portfolio website

JavaScript

Topics: Variables, data types, and basic operators.

Topics: Functions, DOM manipulation, and event listeners.

Topics: Arrays, for and while loops, and array methods.

Topics: let and const, template literals, arrow functions, and destructuring.

JavaScript Projects

Topics: Combine all JS knowledge to add interactivity to your portfolio website.

Task: Add a dynamic skills section that shows progress bars for each skill.

Task: Make a Timer website that have some soothing background and voices with timer running on it

Introduction to Git and GitHub

Topics: Version control basics, cloning, committing, and pushing code to GitHub.

Task: Push your portfolio project to GitHub.

Tailwind

Topics: Using Tailwind CSS for rapid styling.

GSAP

Topics: Learn animation library GSAP.

Introduction to React.js

Topics: React components, JSX, and state.

Task: Create a simple React app that displays your portfolio data dynamically.

Build a Full Project

Topics: Combine HTML, CSS, JavaScript, and React to create a fully functional website.

Project: Build a responsive e-commerce site with product pages, a shopping cart, and a checkout form.
